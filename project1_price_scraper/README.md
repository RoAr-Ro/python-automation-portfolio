# Book Price Scraper

## Description
Script that scrapes all 1000 books from books.toscrape.com
and saves title, price and rating to a CSV file.

## Technologies
- Python 3
- requests
- BeautifulSoup4

## How to run
1. Install dependencies: `pip install requests beautifulsoup4`
2. Run: `python scraper.py`

## Output
- `books.csv` with 1000 books scraped from 50 pages
