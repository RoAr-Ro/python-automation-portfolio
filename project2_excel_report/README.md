# Users Excel Report

## Description
Script that fetches user data from a REST API and generates
a professionally formatted Excel report with styled headers
and auto-adjusted column widths.

## Technologies
- Python 3
- requests
- openpyxl

## How to run
1. Install dependencies: `pip install requests openpyxl`
2. Run: `python report.py`

## Output
- `users_report.xlsx` with 10 users, formatted headers and auto column width
